
# 🐍 SnakeAsia - AI Project

Chào mừng bạn đến với dự án SnakeAsia. Dưới đây là hướng dẫn chi tiết để tải và chạy trò chơi trực tiếp từ Terminal.

## ⚙️ 1. Yêu cầu hệ thống
Trước khi bắt đầu, hãy đảm bảo máy tính của bạn đã cài đặt sẵn:
* **Java JDK** (phiên bản 8 trở lên).
* **Git** 

## 📥 2. Tải mã nguồn (Clone)
Mở Terminal (hoặc Command Prompt/PowerShell) ở thư mục bạn muốn lưu game, sau đó chạy lệnh sau để tải toàn bộ dự án về máy:

```
git clone https://github.com/phanquocuy/SnakeASIA_13.git
```
## 3. Truy cập vào thư mục mã nguồn
Di chuyển vào thư mục dự án và tìm đến nơi chứa các file .java (thư mục src):
```
cd SnakeASIA_13
cd src
```
## 4. Biên dịch mã nguồn (Compile)
Sử dụng trình biên dịch Java để biên dịch toàn bộ các file trong dự án:
```
javac *.java
```
## 5. Khởi chạy Game
Chạy class chính App để bắt đầu trò chơi:
```
java App
```
